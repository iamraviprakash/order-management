## Grocery Orders Dashboard

### Introduction
This is single page application developed using React for listing and searching groceries orders

### Tech Stack
* Material UI: Core & Icons
* React
* React Router
* Redux
* Redux Thunk

### Tools Used
* Figma: UI Design
* GitHub: Version control, App and API hosting
